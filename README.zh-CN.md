# sendfile

[English Version](./README.en.md)

#### 介绍
一个用于临时分享文件的服务器。上传文件，并把链接发送给朋友。上传者可设置分享过期时间、密码、或主动删除文件。

#### 软件架构

* 前端：Vue + Element UI
* 后端：Springboot

#### 安装教程

1. 依赖环境：Java 1.8
1. 到Release页面下载jar文件
2. 运行：java -jar sendfile-x.y.z.jar

#### 使用说明

1. 在浏览器打开：http://localhost:30080
2. 添加一至多个文件
3. 点击“上传”按钮
4. 等待上传结束
5. 复制分享地址
6. 发给朋友
